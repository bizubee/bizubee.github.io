The most intense programming language on Earth!

Bizubee

  • Compiles to modern JavaScript
  • Has built-in support for ES6 modules
  • Integrates seamlessly with JavaScript
  • Can compile to JS and run on the web
  • Can run in a Node environment
  • Is easily extensible and developer friendly
  • Has a static but pluggable module system
  • Has a user friendly compiler core API
  • Supports both curly blocks and indented blocks
  • Supports tree shaking through Rollup
